# Portfolio

Personal portfolio site for [Mikołaj Bień](https://github.com/bieniucieniu).

Built with SolidJS, Vite, Tailwind CSS, and `@solidjs/router`.

## Development

Requires [Bun](https://bun.sh).

```bash
bun install
bun run dev
```

Open [http://localhost:5173](http://localhost:3000).

## Build

```bash
bun run build
bun run serve
```
